Decision Support System

The Decision Support System is related to healthcare of the elderly people. For this, a domain level ontology based on Handbook of Geriatrics was created and then was mapped to an upper level ontology, BFO.

The site has three main parts, Symptom Checker, Disease Lookup and Ontology Browser.

About Project

This project was created to enhance the diagnosis process for elderly people. The site is powered by a Decision Support System based on an Ontology Framework.
